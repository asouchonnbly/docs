---
title: "Rafraîchissement d’un jeton d’accès JWT"
description: "Un jeton expiré ou sur le point d’expirer peut être régénéré via un point d’accès dédié sous condition que le jeton était initialement valide."
api: POST /api/v1/auth/refresh
contentType: application/json
---

```bash cURL
-H 'content-type:application/json
-d '{

https://omnichannel.numberly.com/api/v1/auth/refresh
```
</RequestExample>


## Réponse

Si le jeton est valide, la réponse au format JSON sera de la forme :

<ResponseField name="access_token" type="string"/>

<ResponseField name="expires_in" type="number"/>

<ResponseField name="token_type" type="string"/>

<ResponseField name="refresh_token" type="string"/>
